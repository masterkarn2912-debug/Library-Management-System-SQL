# Library-Management-System-SQL
This project is a Library Management System built using SQL. The goal of this project is to practice working with relational databases and writing SQL queries to solve real-world data problems related to library operations.  The database includes multiple related tables such as books, members, employees, branches, issue records, and return records

Tasks Performed
1. Retrieved and filtered data using SELECT, WHERE, and ORDER BY
2. Connected multiple tables using INNER JOIN and LEFT JOIN
3. Applied GROUP BY, HAVING, and aggregate functions
4. Identified books that were never issued
5. Found members who issued books but did not return them
6. Calculated total rent generated per book

Learning Outcomes
1. Improved understanding of relational database design
2. Gained hands-on experience with joins and aggregation
3.Learned how to analyze data using SQL queries
